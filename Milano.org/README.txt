Questo sito chiamato Milano.org è un sito che abbiamo realizzato io e Tenderini come compito di informatica. 
Con la spiegazione di ajax abbiamo deciso di implementare una parte col l'utilizzo di essa che si occupa di gestire la parte del "contattaci".
Per il funzionamento è necessario creare un database tramite phpmyadmin chiamato musei e importare il file sql fornito. 
Il progetto è stato realizzato con netbeans, quindi è presente la cartella creata da esso chiamata ntbproject.
